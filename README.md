# CodeAlpha Tasks

## Student Grade Management System

A Java-based console application that helps manage student records, store grades, and generate performance statistics such as average, highest, and lowest scores.

### Features

* Add student names and grades
* Display all student records
* Calculate average grade
* Find highest grade
* Find lowest grade
* Generate a summary report
* Menu-driven console interface

### Technologies Used

* Java
* ArrayList
* Object-Oriented Programming (OOP)
* Scanner Class for User Input

## Hotel Booking System

A Java-based console application that allows users to search available rooms, book rooms, view booking details, and cancel reservations with a simple payment simulation.

### Features

* Search rooms by category (Standard, Deluxe, Suite)
* View available room details and prices
* Book hotel rooms
* Simulate payment processing
* View all bookings
* Cancel bookings using Booking ID
* Real-time room availability management
* Menu-driven console interface

### Technologies Used

* Java
* ArrayList
* Object-Oriented Programming (OOP)
* Scanner Class for User Input
